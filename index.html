Loads and parses Elenia consumption hourly export csv file for the year to date (2022). Outputs what electrivity would have cost per month in EUR including VAT with spot marginal. All processing of consumption data is local. Spot prices are loaded until end of August. File format is like this: 
Aikaväli;Keskiteho;Lämpötila
1.1. 00:00:00;0,60;-6,9
1.1. 01:00:00;0,45;
1.1. 02:00:00;0,13;
