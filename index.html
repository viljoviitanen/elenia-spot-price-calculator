Loads and parses Elenia consumption hourly export csv file for the year to date (2022). Outputs what electrivity would have cost per month in EUR including 24% VAT with spot marginal, consumption and average price/kWh.

All processing of consumption data is local. Spot prices are loaded until end of September 2022.

Elenia csv file format is like this: 

Aikaväli;Keskiteho;Lämpötila
1.1. 00:00:00;0,60;-6,9
1.1. 01:00:00;0,45;
1.1. 02:00:00;0,13;
It is found in Elenia Aina service, "Kulutustiedot", "Lataa kulutus tuntitasolla" button.
